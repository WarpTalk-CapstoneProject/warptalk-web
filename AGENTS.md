# Project Agent Instructions

Before starting any coding task, always read and follow:

- `/agents/rules/continuous-workflow-rule.md`

This project uses `/agents/page-docs` as the source of truth for page and feature explanations.

After every code change, update the relevant markdown documentation inside `/agents/page-docs`.

Required workflow:

1. Read `/agents/rules/continuous-workflow-rule.md`.
2. Identify which page, feature, module, or flow is affected.
3. Read the related documentation inside `/agents/page-docs`.
4. Make the code change.
5. Update the related markdown documentation.
6. Mention which documentation files were updated in the final response.

Do not consider the task complete until the related documentation has been checked and updated.
